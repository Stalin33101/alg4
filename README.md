## Лаборатная работа №4
# Сервис сокращения ссылок

## Описание проекта
Упрощённый сервис сокращения ссылок, реализованный на Python в рамках лабораторной работы.

## Функциональность

### Обязательная часть
- Добавление новой ссылки (получает короткий код)
- Получение длинной ссылки по короткому коду
- Проверка существования короткого кода
- Вывод всех сокращённых ссылок

### Вариативная часть (пункт 10)
- **Обратный поиск**: поиск короткого кода по длинной ссылке

## Как это работает

Сервис хранит ссылки в двух словарях:

| Словарь | Назначение |
|---------|------------|
| `short_to_long` | короткий код → длинная ссылка |
| `long_to_short` | длинная ссылка → короткий код (для обратного поиска) |

Короткие коды генерируются автоматически: `"1"`, `"2"`, `"3"` и т.д.
